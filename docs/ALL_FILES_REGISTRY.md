### 🐍 DAG ФАЙЛЫ (3 файла)

**1. etl_postgres_to_multi_db.py** - Основной ETL pipeline
- Размер: 300+ строк Python кода
- Назначение: Ежедневная миграция PostgreSQL → ClickHouse & Cassandra
- Задачи (5):
  1. extract_from_postgres - Извлечение из PostgreSQL
  2. transform_data - Трансформация и валидация
  3. load_to_clickhouse - Загрузка в ClickHouse
  4. load_to_cassandra - Загрузка в Cassandra
  5. verify_data_integrity - Проверка целостности
- Особенности:
  - PostgresHook для подключения
  - Pandas для трансформации
  - Параллельная загрузка
  - Обработка ошибок
  - Логирование
  - Проверка целостности
- Расписание: Ежедневно в 02:00 UTC
- **СТАТУС: ✅ ГОТОВО**

**2. etl_csv_to_databases.py** - CSV загрузка
- Размер: 200+ строк Python кода
- Назначение: Загрузка CSV файлов в ClickHouse & Cassandra
- Задачи (3):
  1. validate_csv_file - Проверка структуры
  2. load_csv_to_clickhouse - Загрузка в ClickHouse
  3. load_csv_to_cassandra - Загрузка в Cassandra
- Особенности:
  - Валидация схемы
  - Обработка кодировок
  - Обработка ошибок
  - Логирование статистики
- Расписание: На спрос (Manual)
- **СТАТУС: ✅ ГОТОВО**

**3. etl_hdfs_to_databases.py** - HDFS/Spark загрузка
- Размер: 250+ строк Python кода
- Назначение: Загрузка Parquet из HDFS через Spark
- Задачи (3):
  1. spark_read_from_hdfs - Чтение из HDFS
  2. transform_spark_data - Трансформация в Spark
  3. load_to_multi_db - Загрузка в оба хранилища
- Особенности:
  - Интеграция с Apache Spark
  - Big Data обработка
  - Парквет формат
  - Оптимизированная загрузка
- Расписание: Еженедельно (пятницы)
- **СТАТУС: ✅ ГОТОВО**
